教师信息管理系统（全栈开发）
项目描述：​ 基于 ASP.NET Web API 和 Vue.js 的前后端分离项目，实现了教师信息的全流程管理。系统包含用户认证、数据管理、批量操作等企业级应用核心功能。
技术栈：
后端：​ ASP.NET Web API, Entity Framework, SQL Server
前端：​ Vue.js, Element UI, Axios
安全：​ Session 认证、MD5 密码加密、验证码机制
核心功能与职责：
设计并实现了用户登录/注册模块，集成验证码校验与密码加密存储
开发了教师信息管理核心模块，支持数据的增删改查、分页、多条件筛选与排序
实现了Excel导入/导出功能，并提供模板下载，支持批量数据操作
完成了批量编辑功能，可对选中记录的指定字段进行一键更新
通过自定义授权过滤器，实现接口级别的访问控制，保障数据安全
项目亮点：
采用前后端分离架构，接口设计清晰，前端组件化程度高。
具备完整的业务数据流处理能力，从用户登录到复杂的数据管理。
注重用户体验，提供了丰富的前端交互，如弹窗、步骤条、状态反馈等。
